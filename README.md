# Do a Barrel Roll — NeoForge 1.21.11

Un mod Minecraft NeoForge **100% natif** qui transforme le vol en élytre en expérience de vol façon simulateur.

## Contrôles en vol élytre

| Entrée | Action |
|--------|--------|
| Souris X | Roll (inclinaison) |
| Souris Y | Pitch corrigé selon le roll |
| A / D | Yaw (lacet) + banking automatique |

## Compilation

Nécessite **JDK 21** et un MDK NeoForge 1.21.11.

1. Copie les fichiers du MDK (`gradle/wrapper/`, `gradlew`, `gradlew.bat`) dans ce dossier
2. Ouvre dans IntelliJ IDEA
3. Lance `gradlew build`
4. Le `.jar` se trouve dans `build/libs/`

## Installation

Copie `barrelroll-1.0.0.jar` dans le dossier `mods/` de ton instance NeoForge 1.21.11.
